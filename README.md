#  Meta Cash

Plataforma de gerenciamento financeiro empresarial desenvolvida para ajudar empresas a organizar, controlar e analisar suas finanças de forma simples, eficiente e inteligente.

---

#  Sobre o Projeto

A Meta Cash é uma plataforma de gerenciamento financeiro com o objetivo de auxiliar empresas no controle de suas entradas e saídas de capital. O sistema oferece visualização clara do fluxo de caixa, gráficos intuitivos, relatórios detalhados e organização das movimentações financeiras, facilitando a tomada de decisões estratégicas.

A proposta é transformar dados financeiros em informações úteis, tornando a gestão financeira mais prática, organizada e acessível para empresas de diferentes portes.

#  Regras de Negócio

* O usuário deve estar cadastrado para acessar o sistema
* Cada usuário deve estar vinculado a uma empresa
* Toda movimentação deve possuir valor, data e categoria
* O valor da movimentação deve ser maior que zero
* O saldo deve ser atualizado automaticamente
* As movimentações devem ser classificadas como receita ou despesa
* O sistema deve manter histórico financeiro
* O usuário só pode visualizar dados da sua empresa
* A exclusão de registros deve exigir confirmação

#  Como Executar o Projeto

## Clone o repositório

```
git clone https://github.com/seu-usuario/meta-cash.git
```

## Acesse a pasta

```
cd meta-cash
```

## Backend

```
cd backend
npm install
npm start
```

## Frontend

```
cd frontend
npm install
npm start
```
